# Carola

To connect to poloniex API.

## Usage

Download and put in your ASDF path, quicklisps local-projects folders is great for that.

```common-lisp
(ql:quickload 'osicat)

(carola:set-api-key your_key_here)
(carola:set-secret_secret_here)

(ql:quickload :carola)

```
